11st October 2023

C, CS50

The time for data structures (lists of pointers in C) has come. Data structure is a term for describing a datas that are storage in memory and are not side by side (as arrays do). It means that we have to add an additional little space to every data and this special space would be used to point to address of another data that we want to keep in the list. This means that in one adres in memory we storage data itself and bounded address to this is a pointer that points to next address etc. Last item of our list keep a NULL value in metadata (that what’s this special additional place called) to let the program know that its end of the list. To start the whole process of listing and pointing we need one additional piece in memory that keep an address of a first element of my list. When this „start“ pointer is being used and the process starts.
