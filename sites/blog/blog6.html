11th April 2023

C, CS50

I completed 3 easy sorting tasks. Sorting isn’t that hard but for now I’m using only few sorting methods. Bubble sort, selection sort and merge sort. Merge sort is the most advanced sorting method I know. It divides the data into smaller groups and sorts these smaller groups by dividing them even more. When it comes to point that divided group contains only 1 value then its compared to to a little bigger group and this comparison means that the values are set in correct order. Later another data group is sorted in the same way as before. That makes the sorting faster but it takes a lot of memory because of all the comparisons and dividing that algorithm takes. Bobble sort and selection sorty are better for smaller data bases. Bubble sort compares two values that are side by side and puts them in correct order. Algorithm runs a number of times switching values and the data is considered sorted when no switches left, that means all values are in correct order. Select sort runs through data base and looks for the smallest value and puts it in first place, then it runs again looking for another smaller value that left in the data base and when its found its settled in correct order. When all values are settled into correct place the data base I considered to be sorted.
