21st February 2024

HTML / CSS, Firebase

I did a Contact tab on my website. Unfortunately, I had problems with making my contact form working correct. At first I have found a PHP solution, I wrote (with help from Internet because I don’t know php yet) a PHP file to send email from page to my mail. But it occurred that when I want to send email the PHP file is downloaded instead of do what it should. I found something about PHP support on Firebase and I started to search for a solution but it seemed to be rather hard to do. So I was looking for another way to do this. I found a tutorial on YT using Firebase realtime database and JS file to initiate it. I have wrote this file (again with help from YT) but and the end, this didn’t work the way I expected. Data was collected on Firebase database but emails weren’t send on my email box. So I was looking for something else. Finally I have found a website formsubmit.co and it was it. First I tried to go with YT tutorial and post my website on Git and manage it from there, but I have found doing it with Firebase more comfortable for me. So I added just simple attributes to my main HTML file, activated formsubmit, deployed all to Firebase and its working perfectly now. Only problem with this site is that is pretty slow, but its free so I can’t explain for now.
